课程介绍：

C4D（Cinema 4D）是德国Maxon Computer研发的3D绘图软件，前身为FastRay，以其运算速度高和渲染插件强著称。

C4D包含建模、动画、 渲染、角色、粒子以及新增的插画等模块，在用其描绘的各类电影中表现突出。而随着其越来越成熟的技术，受到越来越多的电影公司的重视，同时在电视包装领域也表现非凡，成为行业主流软件。
